1. Arrays / HashMap
2. Two Pointers

4. Stack
5. Linked List
6. Binary Search
7. Trees

# Weekly Practice Strategy

```
Week 1 → Solve the Core 15 problems
Week 2 → Solve additional pattern problems
Week 3 → Re-solve Core 15 without looking
Week 4 → Timed practice
```

After this cycle you will recognize **most interview questions instantly**.

---

# Final Advice

For backend interviews (5–10 years experience):

Focus on:

- Explaining your approach clearly
- Writing clean code
- Handling edge cases

Not on solving extremely tricky algorithm puzzles.
